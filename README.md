# Secure Login System Using JavaBeans and MVC

## Description

A secure login system developed using JavaBeans, JSP, Servlets, and the MVC architecture. The application provides user registration, login authentication, session management, and secure database connectivity using JDBC and MySQL.

## Technologies Used

- Java
- JavaBeans
- JSP (JavaServer Pages)
- Servlets
- MVC Architecture
- JDBC
- MySQL
- Apache Tomcat
- HTML
- CSS

## Features

- User Registration
- Secure User Login
- User Authentication
- Session Management
- Logout Functionality
- Database Connectivity
- MVC-Based Architecture

## Requirements

- Java JDK 8 or above
- Visual Studio Code (VS Code)
- Apache Tomcat Server
- MySQL Server
- JDBC MySQL Connector
- Web Browser (Chrome, Edge, or Firefox)

## Project Structure

- JavaBeans
- JSP Pages
- Servlets
- MVC Components
- MySQL Database
